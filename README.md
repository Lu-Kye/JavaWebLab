# How to debug
## PreCondition
* Intellij IDEA
* Configure a `Remote` Run/Debug Configuration in IDEA

Step1: In terminal, run 
```shell 
sh debug.sh
```
Step2: Start debugging `Remote` when You see `Listening for transport dt_socket at address: xxxx` in console.
